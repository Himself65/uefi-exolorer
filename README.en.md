# uefi-exolorer

#### Description
与联想开发隔离计算机之后，有几年的时间，一直在开发UEFI Option Rom。这几年逐渐开始开发其他产品，积累下来的UEFI代码也转交给其他同事维护。国内开发UEFI的人不多，与好友@张佩讨论后，总觉得很可惜。在他的鼓励下，开始撰写这个系列博客-UEFI开发探索，此仓库是用来存储博客中用到的代码的。
